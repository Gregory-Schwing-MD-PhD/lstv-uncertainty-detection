How It Works

This system uses epistemic uncertainty from a spine localizer trained on normal anatomy. When the model encounters LSTV (sacralized L5 or lumbarized S1), it exhibits high uncertainty because the anatomy doesn't match its learned patterns.

Novel Approach: Instead of training a new detector, we repurpose uncertainty from an existing model - a zero-shot detection method!
